### 3. **Create Prisma Schema**

If no `prisma/` directory exists, the addon creates one with a base `schema.prisma` file:

```ts

  provider = "prisma-client-js"
}

  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```
